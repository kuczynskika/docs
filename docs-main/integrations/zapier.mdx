# Zapier Integration (Beta)

Connect MessageDesk to thousands of other apps using **Zapier**.\
Automate workflows by triggering actions when messages are sent or received‚Äîor perform tasks like **creating contacts** and **sending messages** when events occur in other tools.

---

## What You Can Automate

- **When a message is received** ‚Üí Create a helpdesk ticket, post to Slack, or log it to Google Sheets.
- **When a message is sent** ‚Üí Update a CRM, trigger a follow-up task, or notify your team.
- **From your CRM or forms** ‚Üí Automatically create or update contacts in MessageDesk and send personalized texts.

<Note>
  **Requirements:**

  - A [Zapier](https://zapier.com) account and a connected MessageDesk workspace.
  - Outbound texting via Zapier follows the same [**registration**](/carrier-registration/overview), [**delivery**](/messaging/delivery-status-errors), and [**compliance**](/settings/compliance-center) standards as sending directly from MessageDesk.
</Note>

---

## How Zapier Works

Each Zap has two main components:

| Type         | Description                                                                                   |
| ------------ | --------------------------------------------------------------------------------------------- |
| **Triggers** | The events that start your Zap (e.g., a message received in MessageDesk).                     |
| **Actions**  | The tasks Zapier performs after the trigger fires (e.g., send a message or create a contact). |

---

## MessageDesk Triggers

MessageDesk currently supports two triggers in Zapier:

| Trigger              | Description                                                    |
| -------------------- | -------------------------------------------------------------- |
| **Message Received** | Fires when your workspace receives a new inbound text message. |
| **Message Sent**     | Fires when a message is sent from your workspace.              |

---

## Trigger: Message Received

### Description

Fires whenever your MessageDesk workspace receives an inbound text message.

### Example Payload

```json
{
  "id": "01JMB3VDRZ9QTVYVM1S6AHKHRG",
  "from": "+15551234567",
  "to": "+15551237654",
  "text": "Hey, just checking in!",
  "status": "received",
  "createdAt": "2025-02-17T23:28:18Z"
}
```

### Key output fields

| Field                    | Description                                        |
| :----------------------- | :------------------------------------------------- |
| **id**                   | Unique contact ID.                                 |
| **firstName / lastName** | Contact‚Äôs name.                                    |
| **company**              | Associated company name.                           |
| **inboundChannels**      | List of SMS/email addresses linked to the contact. |
| **createdAt**            | When the contact was created or updated.           |

---

## Action: Send Message

### Description

Sends an outbound SMS message from your MessageDesk workspace. Each Zap run sends one message at a time.

### Input Fields

| Field                 | Required | Description                                                |
| :-------------------- | :------- | :--------------------------------------------------------- |
| **To phone number**   | ‚úÖ        | Recipient number (use E.164 format, e.g., `+15551234567`). |
| **Message body**      | ‚úÖ        | Text content of the message.                               |
| **From phone number** | ‚ùå        | Optional; defaults to your workspace‚Äôs primary number.     |

### Example Output

```
{
  "id": "01KBTXJGCB2QT9MKH3M49Y95SF",
  "from": "+15551234567",
  "to": "+15551112222",
  "text": "Hello world!",
  "status": "queued",
  "createdAt": "2025-12-06T22:47:27Z"
}
```

### Key Output Fields

| Field         | Description                                     |
| :------------ | :---------------------------------------------- |
| **id**        | Unique message ID.                              |
| **status**    | Message state (`queued`, `sent`, `failed`).     |
| **from / to** | Sender and recipient phone numbers.             |
| **text**      | Message body text.                              |
| **createdAt** | Timestamp when the message was created or sent. |

<Warning>
  **Warning:** Messages sent through Zapier count toward your MessageDesk message usage and follow the same carrier and compliance rules as messages sent directly from the Inbox.
</Warning>

## Build Your First MessageDesk Zap

### 1. Log in to Zapier

Go to [**zapier.com**](http://zapier.com) and log in or create an account.

### 2. Create a New Zap

Click **Create Zap** on your dashboard.

![Create New Zap Pn](/docs-main/images/create-new-zap.png)

### 3. Add MessageDesk

Search for **MessageDesk (Beta)** and select it.\
Use **version 1.1.0** when prompted.

![Zapier Version Pn](/docs-main/images/zapier-version.png)

You can use MessageDesk as either a zap trigger or as a zap action.

![Trigger Or Action Pn](/docs-main/images/trigger-or-action.png)

### 4. Choose a Trigger

Select one of the following:

- **Message Received**
- **Message Sent**

Then connect your MessageDesk account and select your workspace.

![Zapier Trigger Select Pn](/docs-main/images/zapier-trigger-select.png)

### 5. Add an Action

Choose one of the following:

- **Create or Update Contact**
- **Send Message**

Map your fields, test your Zap, and turn it on.

![Zapier Actions Pn](/docs-main/images/zapier-actions.png)

## Troubleshooting

| Issue                                                   | Solution                                                                               |
| :------------------------------------------------------ | :------------------------------------------------------------------------------------- |
| **Authentication fails when connecting to MessageDesk** | Reconnect your account in Zapier and confirm you‚Äôre logged into the correct workspace. |
| **Message sent but not delivered**                      | Check [Delivery Status & Errors](/messaging/delivery-status-errors) for details.       |
| **Zap not triggering**                                  | Verify your trigger and workspace setup. Send a test message to fire the event.        |
| **Contact not updating correctly**                      | Check field mappings and ensure phone numbers are in E.164 format.                     |
