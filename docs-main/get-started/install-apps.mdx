# Download MessageDesk on Mobile and Desktop

Download the MessageDesk companion on your phone or set it up as a desktop app.

## What's Available on Mobile vs Desktop

**Mobile app features:**

- Access your inbox and conversations
- Send and receive messages
- View and manage contacts
- Update your user profile
- Change inbox appearance (color theme only)

**Desktop-only features:**

- Relays (automations)
- Comments and mentions (coming soon to mobile)
- Full settings access (layout, signatures, workspace settings)

## How to Install MessageDesk as an App on Chrome

Use Chrome to install MessageDesk as a PWA (progressive web app) that looks and behaves like a native app.

### 1. Open **Chrome** and **log in to your MessageDesk workspace**.

- **Click More (three dots)**
- Select the option to **Cast, Save, and Share**

### 2. Select Install Page as App

You can also click the **Install icon in the address bar** (when available).

![Install Pwa Chrome Pn](/docs-main/images/install-pwa-chrome.png)

### 3. Name the App

Name it **MessageDesk**, or give it a Name based on your organization name, etc.

![Install Name Pwa Chrome Pn](/docs-main/images/install-name-pwa-chrome.png)

### 4. MessageDesk will now appear in your OS launcher/dock.

![Pwa Chrome App Installed Pn](/docs-main/images/pwa-chrome-app-installed.png)

<Tip>
  **Tips:**

  - Right-click the app on your taskbar/dock for shortcuts.
  - Manage app settings from the app window: More \> App info \> Settings.
</Tip>

## How to Install MessageDesk as an App on Safari / macOS Sonoma+

Safari can save any site as a standalone web app with separate cookies and history.

### 1. Log in to MessageDesk using Safari

Go to **File** and select **Add to Dock** (or Share button \> Add to Dock).

![Install Safari App Pn](/docs-main/images/install-safari-app.png)

### 2. Name the app and click Add

MessageDesk will save in your **Home \> Applications** and added to the **Dock**.

![Install Safari App Name Pn](/docs-main/images/install-safari-app-name.png)

### 3. Open from Dock/Launchpad and sign in

![Safari App Dock Pn](/docs-main/images/safari-app-dock.png)

<Note>
  **Note:**

  - To change the app name, URL, or icon click: **App menu \> Settings**.
  - **Optional:** You can show/hide navigation controls or match title bar color to site.
  - **Notifications:** Allow in the web app, then manage in **System Settings \> Notifications**.
</Note>

## Quick Troubleshooting

- Can’t find the install option in Chrome?
  - Make sure you’re on the main app URL and using the latest Chrome.
- Safari “Add to Dock” missing?
  - You need macOS Sonoma or later. Update macOS, then try again.
- Beta access issues on mobile?
  - Reopen the TestFlight or Google Play link and ensure you’re logged into the correct Apple/Google account.
