# Search in MessageDesk

Search in MessageDesk is **global**. It spans every module (Inbox, Contacts, and Settings), so you can find what you need without switching context.

You have two search modes:

- **Quick Search:** a fast, keyboard-driven lookup for contacts and conversations.
- **Advanced Search:** a full cross-module search for contacts, conversations, messages, comments, and labels with per-type filters.

---

## Quick Search vs. Advanced Search

| Capability                  | Quick Search (⌘K)                          | Advanced Search                                                       |
| --------------------------- | ------------------------------------------ | --------------------------------------------------------------------- |
| **How to open**             | **⌘K** / **Ctrl K**, or click the search bar | Search icon in Quick Search, **Enter** on a Quick Search result, or the search icon on mobile/tablet |
| **Where it's available**    | Desktop only                               | Desktop, mobile, and tablet                                           |
| **Searches contacts**       | ✅                                          | ✅ (with filters)                                                      |
| **Searches conversations**  | ✅                                          | ✅ (with filters)                                                      |
| **Searches messages**       | ❌                                          | ✅ (with filters, scroll-to-message)                                   |
| **Searches comments**       | ❌                                          | ✅ (with filters, scroll-to-comment)                                   |
| **Searches labels**         | ❌                                          | ✅                                                                     |
| **Recent results**          | Recent conversations before you type       | N/A                                                                   |
| **Pagination**              | N/A                                        | ✅ Per result type                                                     |
| **Best for**                | Jumping to a known contact or thread       | Finding a specific message, comment, label, or running a broad lookup |

<Tip>
  Press **⌘K** to start a fast lookup, then press **Enter** on any result to escalate into Advanced Search at that result.
</Tip>

---

## When to use each

- **Use Quick Search** when you know roughly who or what you're looking for and want to jump straight to it (contact profile or conversation).
- **Use Advanced Search** when you need to find a specific **message**, **comment**, or **label**, or when you want to filter across multiple item types at once.

---

## Mobile and tablet

Quick Search is **desktop-only**. On mobile and tablet, tap the search icon near the account settings icon to open **Advanced Search** directly. It has its own embedded search box.
