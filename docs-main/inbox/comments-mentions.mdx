# Team Collaboration: Comments, Mentions & Emoji Reactions

Work together seamlessly inside MessageDesk without ever sending an unintended text. Use **comments** for internal notes, **mentions** to alert teammates, and **emoji reactions** to respond quickly without typing a message.

---

## Comments

### What are comments?

Comments are **internal-only notes** that appear inside a text thread but are **never sent to contacts**.\
They‚Äôre ideal for:

- Internal handoffs
- Approvals or QA reviews
- Providing context before responding
- Adding reminders or documentation

<Note>
  You‚Äôll know you‚Äôre in **comment mode** when the compose box turns **purple** (default theme).

<Warning>

## Mentions

### What are mentions?

Mentions let you notify teammates directly within a conversation.

- Type **@** to mention one or more teammates (you can even @yourself).
- Mentions trigger **email** and **browser notifications** linking to the conversation.
- Great for follow-ups, approvals, or assigning the next step.

**Example:**\
‚Äú@Jordan please confirm details before closing this ticket.‚Äù

## How to Use Comments, Mentions & Reactions

### 1. Open a conversation

From your **Inbox**, open any conversation.

![Default Inbox Pn](/docs-main/images/DefaultInbox.png)

### 2. Switch to comment mode

Use the **Message Type dropdown** to change from **Message** to **Comment**.\
The composer will turn **purple**.

![Comment Select Pn](/docs-main/images/comment-select.png)

### 3. Mention teammates

Type **@** to search and select teammates. You can mention:

- One or multiple teammates
- Yourself (as a personal reminder)

![Mention Teammate Pn](/docs-main/images/mention-teammate.png)

<Note>
  Mentioning yourself is a great way to set reminders or flag items for follow-up.
</Note>

### 4. Write and send your comment

Type your comment, then click **Send**. It will appear in the thread but **not** be sent as a text.

- Optional: Click **\+** to [attach media](/messaging/media) (images, files, etc.)
- Switch back to **Message** mode to send an actual SMS.

![Comment Published Pn](/docs-main/images/comment-published.png)

### 5. Add an emoji reaction

Hover over any message or comment ‚Üí click the **emoji icon** ‚Üí choose a reaction.\
Click an existing emoji to see who reacted.

![Comment Reactions Pn](/docs-main/images/comment-reactions.png)

### 6. Mentions trigger notifications

Mentioned teammates get:

- **Email notification** with a direct link to the thread
- **Browser push notification** if enabled

![Mention Email Notification Pn](/docs-main/images/mention-email-notification.png)

## Comment Threads

### What are comment threads?

Threads let you reply directly to a specific comment or message‚Äîkeeping related discussions together.

- Organize back-and-forth discussions
- Keep follow-ups separate from the main message feed
- Available on desktop (hover) and mobile (force touch)

![Comment Thread Pn](/docs-main/images/comment-thread.png)

### How to reply in a thread

1. Hover over a comment or message ‚Üí click **Reply**
2. Type your reply
3. Click **Send**

<Note>
  You can reply from the **original comment** or the **last message** in a thread‚Äîboth add to the same thread.
</Note>

![New Comment Thread Posted Pn](/docs-main/images/new-comment-thread-posted.png)

### Best times to use threads

- **Follow-ups:** Ask clarifying questions without cluttering the main chat
- **Status updates:** Keep progress notes in one place
- **Focused discussions:** Avoid cross-talk in busy conversations

## Deleting comments

You can delete comments you no longer need. Comments are **soft-deleted**‚Äîthey are removed from view but remain in the database for record-keeping purposes.

### How to delete a comment

1. Hover over the comment in the conversation thread
2. Click the **trash can icon** (üóëÔ∏è)
3. The comment will be deleted

### What happens when you delete a comment

- **Individual comments** (non-threaded): Disappear from the conversation pane entirely
- **Threaded comments**: A placeholder showing "Comment was deleted" remains to preserve thread context

<Note>
  Deleted comments are soft-deleted. While they no longer appear in the conversation, they are retained in the database and will still appear in **conversation exports**.
</Note>

## FAQ

**Do comments send to the customer?**\
No. They‚Äôre internal-only.

**How do I know I‚Äôm commenting, not texting?**\
