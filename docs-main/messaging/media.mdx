# Send, Receive & Share Multi-Media Files with MessageDesk

<Note>
  **Supported media file types:**

  - .jpg
  - .png
  - .gif
  - .pdf
  - .heic
  - .vcf

  **_File sizes exceeding 500KB automatically send as clickable links._**
</Note>
